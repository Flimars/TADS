### Programação em Banco de Dados.

**Objetivo**
Projetar e implementar funções e procedimentos nos bancos de dados.

**Ementa**
Funções. Estruturas de controle. Procedures. Cursores. Gatilhos.
Tratamento de erros e mensagens.
